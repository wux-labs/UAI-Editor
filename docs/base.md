# 文档编辑

UAI Editor 提供了强大且方便的文档编辑功能。这些功能都放置在“开始”选项卡下。

## 标题设置

选中需要设置的文本，点击工具栏中的标题设置选项，可以设置标题样式。

![](images/base-01.png)

比如，可以将文本设置为一级标题。

![](images/base-02.png)

## 字体设置

选中需要设置的文本，点击工具栏中的字体选项，可以设置字体样式。

![](images/base-03.png)

比如，可以将文本设置为楷体。

![](images/base-04.png)

## 字号设置

选中需要设置的文本，点击工具栏中的字号选项，可以设置字体大小。

![](images/base-05.png)

比如，可以将文本设置为二号。

![](images/base-06.png)
